# 🎮 Tetris

A modern console Tetris clone built purely for fun and practice — no assignments, no marks, just curiosity and learning.

> Goal: sharpen low‑level logic, game loops, and clean C code structure while recreating a timeless classic.

---

## ✨ Highlights

- Classic Tetris mechanics with falling pieces and line clears.
- Console-based UI for a retro, minimal setup.
- Focus on clean, beginner‑friendly C code.
- Built as a practice project to explore game development basics.

## 🧠 Why this project?

We wanted a small but non‑trivial game to:

- Understand how a **game loop** works in C.
- Work with **2D grids, rotation, and collision**.
- Practice writing code that is **modular and readable**, not just “working somehow”.

## 🕹 Controls

(Adjust once controls are implemented.)
